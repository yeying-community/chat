# 如何新增一种翻译？

假设要新增语言 `new`：

1. 复制 `app/locales/en.ts` 为 `app/locales/new.ts`。
2. 编辑 `new.ts`，将 `const en: LocaleType =` 改为 `const new: PartialLocaleType =`，并改为 `export default new;`。
3. 编辑 `app/locales/index.ts`，增加 `import new from "./new.ts";`。
4. 在 `ALL_LANGS` 中加入 `new`。
5. 在 `ALL_LANG_OPTIONS` 中加入 `new: "new lang"`。
6. 翻译 `new.ts` 里的文本内容。
7. 提交 PR，等待维护者合并。
